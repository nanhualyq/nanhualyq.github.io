## 简介
- 功能：模板引擎本质上就是一种文本替换
- 来自：Shopify（一个电商系统）
- 开发语言：Ruby
- 官网：https://shopify.github.io/liquid/

## 从两个形态出发
- {{ }}
- {% %}

## {{ }}
```
# 输出一个变量
{{ page.title }}

# 输出文本之前，用过滤器先处理一下
{{ "/my/fancy/url" | append: ".html" }}
```

## {% %}
### 声明变量
```
{% assign my_string = "Hello World!" %}

{% assign my_int = 25 %}
{% assign my_float = -39.756 %}

{% assign foo = true %}
{% assign bar = false %}
```
### Tag （代码块）

## 抑制空行
`{% assign my_variable = "tomato" %}`
该代码执行后，将会输出一个空行

想要避免这种情况，可以在内部加 `-`。

比如：`{{- -}}` 或 `{%- -%}`
